## Описания

### Security — `com.ugreen.antivirus` 1.16.0.0027
Security uses the ClamAV virus scanning engine to scan system and user files on the device for viruses. You can customize scan policies to regularly scan files on the device, or enable "Real-time protection" to monitor newly added or modified files in real time. Before starting a scan or enabling real-time protection, please ensure that the system has at least 1 GB of available RAM for successful scanning and optimal performance

### Surveillance Center — `com.ugreen.cameramgr` 1.0.0.0677
Surveillance Center is a video recording management app designed for UGREEN NAS users. It supports camera connection and provides capabilities such as live view, video management, event-triggered recording, and notifications. With this app, users can securely store footage from their homes, stores, offices, and other scenarios on NAS devices, making it easier to access live feeds and historical recordings anytime — delivering a safer, more reliable local surveillance management solution

### Comics — `com.ugreen.comic` 1.3.0.0015
The "Comics" app is a powerful and easy-to-use comic reader with an exquisite comic wall layout. It is compatible with massive comic formats and offers various reading settings such as single-page and double-page modes

### DLNA — `com.ugreen.dlna` 1.14.0.0013
After enabling this app, you can play media files from NAS devices on any client device that supports DLNA protocol.

### Docker — `com.ugreen.docker` 1.16.0.0022
Docker is a lightweight virtualization app. You can easily install, run and manage various container apps through Docker, which makes it easy to build personal Web server, database, media server and other apps.

### Downloads — `com.ugreen.downloadmgr` 1.16.0.0024
Downloads supports various types of resources, including HTTP, FTP, BT/PT. For the download of PT resources, this software uses the qBittorrent 4.6.5 download kernel. It is recommended to use it after knowing the rules of PT station.

### TextEdit — `com.ugreen.editor` 1.16.0.0017
TextEdit is a tool for editing plain text content, supporting switch of multiple encoding formats and selection of appearance styles. If you need to view special texts such as html and xml, the TextEdit also supports rich programming languages and corresponding syntax highlighting to improve readability.

### SAN Manager — `com.ugreen.iscsi` 1.9.0.0025
SAN Manager helps you manage iSCSI with ease. As a service in Storage Area Network (SAN), iSCSI provides block-level access, allowing clients to access volume as if it were a local drive

### Virtual Machine — `com.ugreen.kvm` 1.16.0.0020
Virtual Machine (VM) can create and manage virtualization environments. You can easily set and run VMs via this app, such as Linux, Windows and other operating systems. In addition, VMs can flexibly allocate hardware resources so you can run multiple VMs on a single physical server to meet the needs of different applications and workloads.

### Music — `com.ugreen.music` 1.16.0.0007
Music is an app designed specifically for UGREEN NAS users to play music, which aims to help users more efficiently manage their music library, create personalized song lists and share them with friends.

### Cloud Drives — `com.ugreen.netdisk` 1.16.0.0030
"Cloud Drives" is a highly practical and efficient application. It enables users to access their files stored in cloud drives anytime via the "UGREEN NAS" app, and create safe data sync tasks between NAS devices and cloud drives rapidly. Based on user configurations, it effectively meets various data synchronization and backup needs

### Notes — `com.ugreen.note` 1.0.0.0070
An official modern private note app by UGREEN. Easily capture inspirations and securely store your personal knowledge

### Online Office — `com.ugreen.office` 1.16.0.0004
Online Office is a collaborative office app that allows multiple people to edit, collaborate, and share documents online. User data is stored on the local NAS device so it's more secure and reliable than traditional cloud documents. Online office also provides features such as document sharing, permission control, encryption, online comments and annotations, and format conversion.

### Photos — `com.ugreen.photo` 2.1.0.0106
Photos makes it easy to manage all your photos or share moments with your friends.

### Snapshot — `com.ugreen.snapshot` 1.10.0.0022
Snapshots provide an efficient data protection solution by backing up data at specific points in time and allowing for quick restoration to any historical state. It safeguard your data against risks such as accidental deletion, software failures, data corruption, and virus attacks.

### Sync & Backup — `com.ugreen.syncbackup` 1.16.0.0047
Sync function supports syncing data between the UGREEN NAS and PC/remote UGREEN NAS/file server to ensure data sync between different devices. 
Backup can help you protect your data from accidental loss. The backup target can be the local UGREEN NAS/file server/PC/mobile phone, and backup tasks can be set according to user needs.

### Vault — `com.ugreen.vault` 1.16.0.0032
Effectively protect sensitive data in "Vault" through encryption and access control
▪ When enabling the app, other administrators of the device will have exclusive private space
▪ Each administrator's data will be stored independently and cannot be accessed or managed by others

### File Version Explorer — `com.ugreen.versionmgr` 1.16.0.0019
File Version Explorer is used to record and manage the version history of all files on the NAS. Every change is traceable, eliminating version chaos and improving file management and collaboration efficiency.

### Theater — `com.ugreen.videomgr` 1.16.0.0027
Theater is a video media service app created for UGREEN NAS home users, which is committed to providing users with excellent multimedia experience. Through Theater, users can easily manage their own video content, build poster walls, and conveniently watch their favorite movies, TV series and other video content in various devices.
